Fonctionnalités Clés

Espace Personnel

Un tableau de bord client complet permettant de modifier les informations personnelles et de gérer les avis soumis.

Prise de Rendez-vous

Les utilisateurs connectés peuvent envoyer des demandes de rendez-vous détaillées. Le praticien et le client sont notifiés par e-mail.

Gestion des Avis

Les clients peuvent soumettre un avis pour des prestations spécifiques.

À Propos de ce Projet

Ce projet a été inspiré par le besoin concret d'une professionnelle de confiance de disposer d'une visibilité en ligne. L'objectif était de créer un MVP (Minimum Viable Product) robuste en 4 semaines, en utilisant Python et Flask-RESTX pour le back-end et HTML, CSS et JavaScript pour le front-end. Le tout en conteneurisant l'application avec Docker et en intégrant les contraintes de gestion des données (RGPD).

Ce projet a été développé dans le cadre de ma formation de développeur web et web mobile à Holberton School.
