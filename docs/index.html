Key Features

Personal Space

A comprehensive customer dashboard allowing you to modify your personal information and manage submitted reviews.

Appointment Booking

Logged-in users can send detailed appointment requests. Both the practitioner and the client are notified by email.

Review Management

Customers can submit reviews for specific services.

About this Project

This project was inspired by the real need for a trusted professional to have online visibility. The goal was to create a robust MVP (Minimum Viable Product) in 4 weeks, using Python and Flask-RESTX for the back-end and HTML, CSS, and JavaScript for the front-end. All while containerizing the application with Docker and integrating data management constraints (GDPR).

This project was developed as part of my web and mobile web developer training at Holberton School.
